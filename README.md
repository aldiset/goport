# goport
Tools to scan port with golang

## install golang
- ```https://go.dev/doc/install```

## Run
- ```go run goport.go --ip 192.168.x.x --port 1-1000``` or
- ```go run goport.go --ip 192.168.x.x --port 21,22,23,25,80```
